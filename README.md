Ejercicio Github

Este es el ejemplo en clase pero completado. Se puede realizar las consultas de:
	LISTAR repositorios
	GET un repositorio a partir de su nombre
	POST un repositorio
	PUT repositorio
	DELETE repositorio

Para poder realizar la prueba, se debe inicar desde el archivo 'Princpial.html' que permite navegar por el resto de páginas y así realizar las consultas.
